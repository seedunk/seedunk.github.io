# Flutter
  > 只要一套代码库，即可构建、测试和发布适用于移动、Web、桌面和嵌入式平台的精美应用。
## 开发环境
### 在Visual Studio Code中进行开发
1. 安装插件 [https://marketplace.visualstudio.com/items?itemName=Dart-Code.flutter](https://marketplace.visualstudio.com/items?itemName=Dart-Code.flutter)。
2. 初始化项目 `npx vitepress init`。根据向导配置相关项。
